# Player Auction System

A web-based application that simulates a player auction system similar to the IPL auction, allowing users to create teams, bid on players, and manage squads.

## 🚀 Features

- Create and manage teams
- Real-time player auction and bidding
- Admin dashboard to manage players and teams
- Authentication for users and admins
- Responsive design for all devices

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)

## 📦 Installation

1. Clone the repository:

   git clone https://github.com/Jayanth175/player-auction-system.git
   cd player-auction-system

2. Install dependencies:

    npm install

3. Create a .env file:

    MONGO_URL=mongodb+srv://<your_mongodb_connection_string>
    JWT_SECRET=<your_jwt_secret>

4. Run the development server:

    npm start

The server will start on http://localhost:3000
